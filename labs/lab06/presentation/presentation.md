## Формат `pdf`

- Использование LaTeX
- Пакет для презентации: [beamer](https://ctan.org/pkg/beamer)
- Тема оформления: `metropolis`

- Используется [тема](https://revealjs.com/themes/) `beige`

- Тема задаётся в файле `Makefile`

```make
REVEALJS_THEME = beige 
```
